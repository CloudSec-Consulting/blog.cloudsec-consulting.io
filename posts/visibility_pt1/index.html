Introduction

In this lame attempt at getting started with tech blogging, I wanted to share my journey of using different cloud providers. Probably not like the first users of public clouds, this journey started with Google Cloud Platform and not with AWS or Azure.

After the first few months of discovering the world of GCP, different projects landed on either AWS, Azure or GCP. Although my experience at using Azure at this moment is very limited, I hope I get to use their services asap, even if they are going through a bad period since a lot of disturbing security issues are discovered. Let me get this clear, the problem is not with discovering security vulnerabilities, in this case the problem is more with the way Microsoft are handling the communication around the issues, which is none at the time of writing.

Enough Microsoft bashing at the moment and let’s get started with this blog series. Yep, I’m planning it as a multi part small article, first because I can’t stand long blogs, and second I will be addressing different tools and step-by-step guides; so having them in separate blog posts will keep thing clean and tidy.
